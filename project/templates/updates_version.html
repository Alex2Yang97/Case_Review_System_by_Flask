8.21 Version Updates
  • Backend Migration: We've successfully transitioned from SQL Server to SQLite, a change that not only boosts the application's performance and agility but also simplifies its setup and maintenance.

  • User Access Control: In response to user feedback, we've expanded the range of user access functions, allowing us to tailor permissions according to different roles and responsibilities.
  • Bug Fixes: We've diligently addressed bugs that were previously reported by users, ensuring a smoother experience with this updated version.
